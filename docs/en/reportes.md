# Reports screen

The reports screen allows viewing, downloading, and managing all reports generated during an analysis.

![Reports](/images/reportes.png)

## SAES reports

- Friedman
- MeanMedian
- Wilcoxon
- WilcoxonPivot

## Download all reports

If reports are available, the following button will appear:

**Download all**

This button automatically generates a ZIP file containing all reports associated with the analysis.

# Search bar

The search bar allows quickly locating specific reports.

You can search by:

- Filename.
- Report type.
- Extension.

Search results are updated automatically.

Allows selecting two reports to compare them easily.
