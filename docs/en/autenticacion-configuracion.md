# Authentication and settings

This section explains how to access the application, create an account, and manage user settings.

---

![Main screen](/images/inicio.png)

## Create an account

To use the application, you need to create an account with a username, email address, and password.

::: tip Password requirements
The password must contain at least:

- 8 characters.
- One uppercase letter.
- One lowercase letter.
- One number.
:::

### Registration steps

1. On the main screen, click **Sign up**.
2. Enter a username.
3. Enter a valid email address.
4. Enter a secure password.
5. Click **Create account**.

If everything is correct, you will be able to log in with your new account.

<video src="/videos/registro.mp4" controls muted loop playsinline class="docs-video"></video>

---

## Log in

Once the account has been created, you can access the application from the login screen.

### Login steps

1. Click **Log in**.
2. Enter your username.
3. Enter your password.
4. Click **Sign in**.

::: info
If the credentials are correct, you will automatically access the private area of the application.
:::

![Login screen](/images/iniciar_sesion.png)

---

## User settings

To access the settings page, click **Settings** from your profile menu.

From this section you can modify the main information of your account.

![Settings](/images/configuracion.png)

## Available options

### Change username

1. Locate the username option.
2. Enter the new username.
3. Save the changes.

::: warning
The new username must be unique.
:::

### Change email address

1. Locate the email option.
2. Enter the new email address.
3. Save the changes.

![Change profile information](/images/perfil.png)

### Change password

1. Enter the current password.
2. Enter the new password.
3. Save the changes.

![Change password](/images/password.png)

### Delete account

::: danger Warning
Deleting the account will also remove all associated analyses, datasets, and generated results.
This action cannot be undone.
:::

![Delete account](/images/delete.png)
