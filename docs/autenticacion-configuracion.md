# Autenticación y configuración

En esta sección se explica cómo acceder a la aplicación, crear una cuenta y gestionar la configuración del usuario.

---

![Pantalla principal](/images/inicio.png)

## Crear una cuenta

Para usar la aplicación necesitas crear una cuenta con un nombre de usuario, correo electrónico y contraseña.

::: tip Requisitos de la contraseña
La contraseña debe tener al menos:

- 8 caracteres.
- Una letra mayúscula.
- Una letra minúscula.
- Un número.
:::


### Pasos para registrarse

1. En la pantalla inicial, pulsa **Registrarse**.
2. Introduce un nombre de usuario.
3. Introduce un correo electrónico válido.
4. Introduce una contraseña segura.
5. Pulsa **Crear cuenta**.

Si todo ha ido bien, ya podrás iniciar sesión con tu nuevo usuario.

---

## Iniciar sesión

Una vez creada la cuenta, puedes acceder a la aplicación desde la pantalla de inicio de sesión.

### Pasos para iniciar sesión

1. Pulsa **Iniciar sesión**.
2. Escribe tu nombre de usuario.
3. Escribe tu contraseña.
4. Pulsa **Entrar**.

::: info
Si las credenciales son correctas, accederás automáticamente a la zona privada de la aplicación.
:::


![Inicio de sesión](/images/iniciar_sesion.png)
---

## Configuración de usuario

Para acceder a la configuración, pulsa en **Configuración** desde tu perfil.

Desde esta sección puedes modificar los datos principales de tu cuenta.

![Configuracion](/images/configuracion.png)

## Opciones disponibles

### Cambiar usuario
1. Busca la opción de nombre de usuario.
2. Escribe el nuevo usuario.
3. Guarda los cambios.

::: warning
El nuevo nombre de usuario debe ser único
:::

### Cambiar correo electrónico
1. Busca la opción de correo electrónico.
2. Introduce el nuevo correo.
3. Guarda los cambios.

![Cambiar datos del perfil](/images/perfil.png)

### Cambiar contraseña
1. Introduce la contraseña actual.
2. Escribe la nueva contraseña.
3. Guarda los cambios.

![Cambiar contraseña](/images/password.png)

### Eliminar cuenta

::: danger Atención
Eliminar la cuenta borra también los análisis, datasets y resultados asociados.
Esta acción no se puede deshacer.
:::

![Eliminar cuenta](/images/delete.png)
