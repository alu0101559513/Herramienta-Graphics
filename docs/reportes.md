# Pantalla de reportes

La pantalla de reportes permite visualizar, descargar y gestionar todos los reportes generados durante un análisis. 

![Reportes](/images/reportes.png)

## Reportes SAES

  - Friedman
  - MeanMedian
  - Wilcoxon
  - WilcoxonPivot

## Descargar todos los reportes

Si existen reportes disponibles, aparecerá el botón:

**Descargar todo**

Este botón genera automáticamente un archivo ZIP con todos los reportes asociados al análisis.


# Buscador

La barra de búsqueda permite localizar rápidamente reportes concretos. 
Puedes buscar por:

- Nombre del fichero.
- Tipo de reporte.
- Extensión.

La búsqueda se actualiza automáticamente.
