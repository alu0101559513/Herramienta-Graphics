# Pantalla de inicio

![Pantalla principal](/images/home.png)

## Qué puedes hacer

Desde aquí puedes:

- Acceder a la sección de análisis.
- Consultar los análisis recientes.

![Funcionalidades pantalla principal](/images/home2.png)

## Barra de navegación

La barra de navegación superior permite acceder rápidamente a las principales opciones de la aplicación.

Desde esta barra puedes:

- Acceder a la sección análisis.
- Cambiar el idioma de la interfaz.
- Activar el modo oscuro.
- Abrir la configuración del perfil.
- Cerrar sesión.
